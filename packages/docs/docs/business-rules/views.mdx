# Inbox element views in dialogporten-frontend

Inbox elements (`dialogs`) are filtered in different views, according to status and/or other modifiers, reflecting their state somehow.

## Mapping (WIP)

| Status (enum) | Other Conditions             | Outcome                  |
|---------------|-----------------------------|--------------------------|
| NEW           | lastModifiedBy=tjenesteier | innboks                  |
|               |                             | innboks                  |
| IN_PROGRESS   |                             | utkast                   |
|               | lastModifiedBy=bruker      | utkast                   |
|               | SIGNING                     | utkast                   |
| WAITING       |                             | sendt                    |
| COMPLETED     |                             | sendt                    |
| Cancelled     | lastModifiedBy=tjenesteier | slettet                  |
|               | label:"slettet"*            | slettet                  |
|               |                             |                          |
|               | label:"arkiv"              | arkiv                    |
|               | label:*                             | label:"*"   |

* `label` property on `GETDialogDTO `
